主流平台封面图尺寸速查

2024-08-24
2 分钟阅读时长
札记

问：投稿 12 个平台需要几张封面图？ 答：4 张。太好了居然不是 12 张XD。

可惜各家平台的 UI 比较混乱，总是因为小细节打回给动效老师调整，遂整理了一份基础规范。出于尊重知识产权考虑，无法公开完整表格。但为了防止以后重复造轮子，遂将部分基础尺寸要求留档。

我对各平台封面的基本要求是「在任意 UI 界面裁剪下，封面中的核心信息传递准确」，一般在平台投稿界面，会有基本的尺寸说明。但是，是的总要有但是，实际在平台的不同界面下，封面裁剪尺寸各异。这说白了还挺琐碎的，遍历各个界面确认裁剪比例，绘制出核心显示范围，将信息放置在「核心展示区」即可。

该说不说，b 站的 UI 在网页端、移动端的不同界面都不统一，甚至出现过 16:10 这种在投稿页面封面要求中没有指明（但实际存在）的裁剪尺寸，怒斥😡。

公众号的尺寸正常是 2.35:1，但考虑到也有 1:1 的裁剪展示，把两个尺寸拼接在一起 3.35:1，利用平台自带的裁剪框选择，即可在两个尺寸在均保证信息完整。

同家的视频号就比较抓马了……封面尺寸是随视频尺寸变动的，对于横版视频和竖版视频，封面的裁剪需求都不一致；在不同界面下，裁剪的尺寸也比较横跳。

另外，现在短视频平台（抖音、小红书、快手）投稿横版视频时，可上传两张封面（通常为 16:9 和 3:4）。在移动端信息流展示中，3:4 的尺寸占比更大，故投稿横版视频时需同步上传 3:4 封面。

另附说明（以 b 站尺寸为例）：

  1. 多比例尺寸 16:10 的兼顾要求，出自主页视频和排行榜上展示比例。若上传多比例封面图，则 B 站默认从 16:9 中截取 16:10。另，16:9 兼顾 4:3 的单图中实则已含 16:10 要求。

  2. 核心展示区参考图中，UI 组件在实际展示中为半透明遮罩/数字/图标，存在一定遮挡而非硬性裁剪，但建议不要放置大块文字，会影响观众辨识。

  3. 核心展示区参考图中，榜单排名为实心遮罩，建议不要放置关键信息。

  4. 联合投稿视频需注意，右上角会显示合作标识，此处为实心遮罩，建议不要放置关键信息。

  5. 若视频上热门或为联合投稿，则右上角会出现实心遮罩标识；若两者同时出现，则会出现两个标识。

  6. 各部分优先级由高到低如下：比例裁剪 > UI 组件 > 榜单排名 > 合作/热门标识 > 合作+热门标识，若构图无法兼顾可酌情忽略优先级较低的部分。

  7. 做好封面后最好缩小到 200×100pt 左右，检查是否能看清关键信息，若看不清则需再度调整。也可自行进入投稿页面，使用平台提供的预览效果自查。
