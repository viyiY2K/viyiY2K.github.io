广告传播与 7M 框架

2025-09-25
4 分钟阅读时长

媒体使用与数字广告趋势

本模块讨论目标定位与信息传播，也就是营销中与广告创意密切相关的部分。内容首先介绍不同媒体的使用趋势，随后以经典的牛奶营销活动为例，说明营销传播的 7M 框架；在此基础上，再讨论理性诉求、情感诉求以及社交媒体营销。

消费者接触的媒体并不只有电视，还包括广播、手机、互联网和印刷媒体。相关数据来自美国，但这些趋势在世界其他地区也具有一定的普遍性。总体而言，人们每天会花费大量时间接触数字媒体、电视、广播和印刷媒体。

从 2010 年到 2013 年的预测数据看，电视使用时间相对稳定。即使电视的呈现形式发生变化，或人们在观看电视的同时处理其他事情，电视仍然是重要的媒体形式。

各种媒体的广告支出也在增长，其中电视仍处于领先位置。与此同时，互联网广告、展示广告和移动营销等形式预计将继续增长。后续内容还会结合相关研究，进一步讨论不同媒体与技术的特点。

“Got Milk?” 活动背景

“Got Milk?” 是一项始于 1992 年的美国经典营销活动，推广的产品是牛奶，发起方是加利福尼亚州乳品加工委员会。1987 年至 1992 年间，牛奶消费量明显下降，因此委员会希望扭转这一趋势。

调查发现，大多数人已经认为牛奶有益健康，也认为牛奶的味道不错。因此，活动并不是要改变消费者对牛奶本身的基本评价，而是希望促使人们在家中储存更多牛奶，从而增加消费量。

这背后体现了一个常见现象：当家中现有的产品数量较多时，人们往往会使用得更多。例如，冰箱里存有一箱 12 瓶百事可乐时，消费者可能会比库存有限时喝得更多；当供应有限时，人们则会开始控制消费。

如果希望进一步了解这一现象，尤其是它与食物消费之间的关系，可以参考布赖恩·万斯金的研究及其著作《无意识饮食》。

匮乏诉求与创意设计

乳品加工委员会参与制作的一则广告，围绕“不要让家里的牛奶用完”展开。广告通过缺少牛奶可能带来的不便，强化在家中保持牛奶库存的重要性。

广告开头出现了一位令人反感的男子。他身处美国某个大城市，一边打电话，一边大声解雇某个人；随后，他被一辆卡车撞倒，但故事并没有在这里结束。课程没有直接揭示广告结尾，而是让学习者自行观看并观察后续情节。

这一创意运用了匮乏或剥夺诉求：它不是单纯说明牛奶的功能，而是让消费者设想没有牛奶时会发生什么。

7M 传播规划框架

这项活动对加利福尼亚州乳品加工委员会而言取得了良好效果，也可以用营销传播的 7M 框架进行分析。

Market：目标市场

目标受众是当前已经饮用牛奶的人，而不是从不喝牛奶的人。让不饮用牛奶的人改变习惯可能更困难，其中一些人还可能因为健康原因不喝牛奶，因此活动聚焦于现有消费者。

Message Content：信息内容

传播的核心信息是：家中应当备有足够的牛奶，避免牛奶用完。

Mission：传播任务

活动希望增加牛奶消费量。如果家中的牛奶库存更多，消费者也许每周会多喝一杯。

Message Design：信息设计

创意方案围绕“剥夺”展开，让消费者设想牛奶用完以后生活中可能出现的不便，并将这种情境表现为一场灾难。

Media Strategy：媒体策略

活动使用了电视和印刷媒体。需要注意的是，这项活动开展于 1992 年，当时的媒体环境与今天不同。

Money：传播预算

课程没有给出准确的预算金额，但可以判断，为实施这项活动投入了相当可观的资金。

Measurement：效果衡量

传播效果的衡量至关重要。约翰·华纳梅克在 1928 年提出，广告投入中可能有一部分被浪费，而难点在于无法判断究竟是哪一部分。对于一项假设投入 3,000 万美元的广告活动，广告主尤其需要知道活动是否有效，以及如何衡量其有效性。

7M 框架的应用问题

使用 7M 框架分析一项传播活动时，可以依次提出以下问题：

Market：与谁沟通

目标细分市场是谁？活动希望影响哪些人？

Message Content：传达什么

活动要表达什么信息？应当如何进行定位？

Mission：实现什么目标

传播希望受众产生怎样的思考、感受或行动？

Message Design：如何表达

创意更侧重情感诉求，还是理性诉求？

Media Strategy：通过什么媒体传播

活动是本地广告、电视广告、户外广告牌，还是采用其他传播工具？

Money：投入多少

活动需要多少预算？预期能够获得怎样的投资回报？

Measurement：如何判断效果

应当采用什么指标和方法评估活动是否实现目标？

广告效果衡量

广告效果可以从两个层面衡量。第一层是受众反应，例如人们是否喜欢、记住并理解广告；第二层是实际结果，例如销量是否增加、消费者偏好是否提升，以及人们对品牌或产品的看法是否发生改变。

本课程会把主要篇幅放在 Mission 和 Message 上。其余五个 M 的具体选择往往与所讨论的产品类型密切相关，需要结合实际情境判断。
