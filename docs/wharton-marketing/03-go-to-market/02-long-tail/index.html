长尾理论与数字渠道

2025-09-25
13 分钟阅读时长

长尾曲线中的头部与尾部

长尾理论是理解线上线下渠道结合的重要概念，由克里斯·安德森提出并在其著作中系统阐述。它有助于解释互联网如何改变销售、购物与购买行为。我们将从互联网出现之前的情况的背景知识开始。然后，本节介绍“长尾理论”及其相关经济学。接着，本节介绍麻省理工学院一些同事的研究，他们研究了全渠道和长尾理论的应用。

最后，本节介绍我自己的研究，我称之为“空间长尾理论”。下文将进一步解释这一概念。因此，先从起点开始，先看看所谓的长尾图。这将会出现很多次。因此，长尾有两个轴，一个是 y 轴，另一个是 x 轴。y 轴是衡量流行度的指标，在许多情况下，这将是产品的销量。它也可能是下载次数，比如跟随某人在 Twitter 上的人数。这只是流行度的一个衡量标准。在 x 轴上，所有产品从最受欢迎的产品排列到最不受欢迎的产品。以一个我们都能理解的常见例子来说明。可以想象在 amazon.com 上所有可供销售的书籍。

假设可能有五百万本书可供销售。具体数字并不确定，但这是一个巨大的数字，肯定比任何实体零售商能卖出的书籍数量都要多。假设销量最好的书是《哈利·波特与魔法石》。人们似乎都喜欢哈利·波特。因此，假设这是最畅销的书。并且，在长长的尾巴的末端，在第3,900,000个排名上，可能存在一些关于，费城园艺或其他类似主题的非常冷门的书。因此，长尾理论的想法是，那些在右侧橙色区域边缘的产品非常不受欢迎。或者非常细分，周转速度慢，对于我们俩来说，开设一家实体书店来销售它们根本就不合理。

这就是其基本含义。另一方面，头部产品，如果我们有实体店，我们希望库存的就是这些产品，因为它们是销售量最大的。这就是头部和尾巴的区别，长尾理论的关键思想是，互联网使我们能够访问尾巴中的所有产品。实际上，根据你是什么样的卖家，从尾巴中的所有产品中来的总销售额和利润，真的可以累积到相当可观的数量。这是一个基本的描述。

长尾形成的供给侧与需求侧原因

下面继续讨论一些背景和历史。因此，从历史的角度来看，至少在互联网之前，我们都生活在一个所谓的热门产品的世界里。大片、大片电影、大片、大片音乐专辑等等。大多数人最终都是通过购买和消费流行的事物来结束，公司也被激励去尝试制作那些大片。然而，由于互联网，分销的经济模式已经发生了很大的变化。现在可以提供比以往任何时候都要多的选择。因此，现在我想从两个不同的角度来探讨长尾现象，即供给方面和需求方面。这两个角度在考虑市场时都很重要。因此，在供给方面，由于互联网，卖家现在可以提供更多的种类，无论是啤酒、尿布、鲜花、电影、歌曲，还是其他任何东西。

可以提供更多的种类，因为分销这些产品的经济模式已经发生了巨大的变化。其次，对于长尾现象，也存在一个需求方面的解释，或者与之相关的需求故事。那么在以前，我如何了解一些不寻常的产品和服务呢？可能通过朋友和家人，或者我遇到的人，熟人。但是，本质上我可能因为零售商不会销售这些产品而无法理解或遇到这些产品。记住，零售商或商店只会真正销售那些在人们心中的产品，因此找到尾部产品非常困难。因此，这一点的含义是，我们的口味现在变得更加多样化，因为，简单地说，我们能够接触到更多的需求。

也就是说，接触到更多种类的产品，比我们以前任何时候都多。

长尾经济学与经典分布规律

因此，现在让我比较和对比一下我们接下来将要探讨的长尾经济学的经济学原理，与那些仍然相当有用但已经存在很长时间的老经济学理念。有位名叫帕累托的人，在很久以前，也就是1897年，注意到英国居民和公民的80%的财富只被20%的人拥有。最近我们发现，我们获得的80%的利润是基于我们销售的20%的产品，或者20%的客户。因此产生了常见的80/20法则。现在，长尾效应的某些变化规则有所改变，因为它的比例不再是80/20，我们可以拥有一个巨大的产品长尾，这些产品数量众多，但单个产品的利润实际上非常少。

但总体上却占有了很大一部分。因此，下面继续看看这是如何运作的。另一个有趣的定律是以乔治·齐夫的名字命名的，齐夫定律。他发现，如果你取一份手稿或一本书，书中第二受欢迎的词或最常用的词的频率大约是最受欢迎词频率的一半。而第三常见使用的词的频率大约是最受欢迎词的三分之一。这个定律也适用于其他事物，如一个国家内人口分布等。因此，这里有两个关键原则在幻灯片上，它们真正地支撑了长尾效应这一概念，也将在我们讨论偏好隔离时再次出现。

地域性限制与利基市场

那么，先从第一个开始，我在幻灯片上提到的这个概念是“地域性暴政”。这是什么意思呢？

这意味着你当地可用的东西是由当地卖家决定提供给你的。他们可能与您的口味不同，因此在我们更深入地探讨长尾之前，我想先问大家两个问题。一个与供给相关，另一个与需求相关。因此，可以思考，一个您使用或喜欢的产品，但您在当地找不到，因为它在您的邻居中并不流行。想象一下，您喜欢晒太阳和去海滩，您喜欢使用一种您找不到的特殊乳液，因为该地区的药店只提供非常高的 SPF 产品，而这并不是您想要的。因此，试着想一下您在当地找不到的产品。这将是长尾需求侧的解释，其中所有这些多样性都将在网上提供。

其次，可以思考一下，如果您是供给侧的卖家，您有一件不寻常的产品，您会在哪里想要卖这个产品。提到的产品在 Chris 的书中，实际上有一部非常好的电影叫做《印度往事》。现在，如果我想在美国的物理屏幕上展示，我可能不得不去像洛杉矶和纽约这样的大城市，那里人口密集，或许有很多来自印度的人，或者对印度文化有亲和力的人，我会在那里展示。不可能在全国不同市场的两三个人面前展示。但现在通过互联网，我可以让它可供下载，这里两个人看，那里三个人看，这里四个人，那里五个人，这样需求就会很大。

因此，希望你们能通过这两个例子，以及你们思考的那个例子，看到长尾既满足了产品的需求。现在你们可以得到这种多样性。它还影响了产品的供应方式。回到我们的图表。我将在图表上再提出两个观点。

下面有两个问题，我将引导大家思考。这里是红色部分表示头部，橙色部分表示尾部。

首先需要回答一个问题，然后我会公布我的答案。您对从头部或尾部随机抽取的产品满意吗？可能是一部来自那个位于赤道以南的伟大小国的精彩电影，叫什么名字？不是澳大利亚，另一个，在右边的那一个，新西兰，叫做《指环王》。这是一部相当流行的电影，如果您随机挑选，您可能会非常满意。您可能会喜欢它的八成。相反，如果我抽取了一部非常冷门、不流行的电影，一个来自尾部的利基电影。我向您展示了这一点，您可能不太喜欢它。因此，总的来说，从头部到尾部，平均满意度会下降。

但这只是平均数，正如您从皮特那里学到的，方差也非常重要。然而，可能存在一个在尾部的产品，大多数其他人不喜欢，但对您来说是一部十全十美的电影、歌曲、书籍或其他您正在寻找的东西。因此，如果您有一个高效的方法来过滤并进入长尾，您就能找到真正喜欢的东西。这就是其基本含义。因此，平均来说，当您从左到右移动时，如果您被展示一个随机产品，您的满意度会下降，但如果您能够进入右侧并找到您确切想要的东西，您的满意度可能会非常高。因此，下面继续这些想法。我们这里的长尾有哪些主要特征？

首先，利基产品销售与大热产品销售的比例正在变化。现在，来自所有这些利基产品的需求越来越多。分布效率也在变化。现在可以通过平板电脑和手机特别访问内容类型产品。在能够找到进入长尾的方法方面，消费者评论等就发挥作用了，我们将在后续讨论中更多地谈论评论。因此，正如我在幻灯片上所展示的，长尾只是不受稀缺性文化过滤的。因此，在以前，商店的老板，也就是在这里做视频的我们的朋友 Chris，他会决定在当地市场上提供什么商品。他会决定在屏幕上放映什么电影。

他会决定在他的酒类商店里提供什么啤酒。但现在，通过长尾效应，所有这些都可以通过互联网提供。

互联网渠道与目录渠道的比较研究

因此，到现在我们可能都已经很好地理解了长尾理论的基本概念。因此，我想通过一些在麻省理工学院同事进行的一项非常有趣的研究来展示其应用。

下面说明这里的情况。这些同事想要尝试理解或解开长尾的供给侧有效性的问题，即相对于需求侧，可供选择的多样性要多得多。长尾的解释是，现在你更容易搜索并找到真正喜欢且符合你口味的东西，那些在长尾橙色部分的产品。因此，麻省理工学院的这项研究作者所做的是，他们获取了一家大型零售公司的销售数据，该公司通过两个不同的渠道进行销售，一个是互联网渠道，另一个是通过目录渠道。他们想看看这两个渠道是否有着相同的销售组合。现在，这两个渠道的价格和产品都是相同的，但他们想问的问题是，这些渠道在交付产品和客户的方式上是否存在任何差异？

现在让我来介绍一个图表。这实际上是一个相当酷的想法，这一图表可能并不陌生，但可能不是在这个背景下。我这里展示的是一个图表，我们可以用它来计算一个叫做基尼系数的东西，基尼系数，这就是幻灯片顶部写的内容。基尼系数等于面积 A。除以面积 A 加上面积 B。

下面说明这个图表中发生的事情。这通常是一个用来理解一个国家内收入平等或不平等性的图表。但它也可以用来理解一个特定公司销售的各种产品之间的销售平等或不平等性。为了帮助我们理解这一点，下面来看图表上的点 C。点 C 位于 X 轴的 0%到 100%之间大约一半的位置。

这意味着，如果我们将其应用于一个国家的收入，如果我们取收入最低的人群，比如在美国，一直到最后达到 50%的收入水平，然后我们从 C 点与曲线相交的地方画一条虚线，一直延伸到右侧，在我看来，这看起来像是 Y 轴的五分之一左右，或者说 20%。因此，这意味着在这个特定国家中，最底层的50%的人拥有大约20%的所有收入。因此，这是一个收入分配稍微不均的国家。现在可以想象，如果洛伦兹曲线更接近图表的远右端点，那么这将是一个所有财富都掌握在极少数人手中的国家。同样，如果你考虑产品，如果你将那个圆圈或半圆圈缩小，那么这将是一家几乎所有销售额都来自一到两个产品的公司，这就是这个概念。

因此，当研究的作者们观察这一点时，他们发现，与目录渠道相比，互联网渠道的基尼系数更为分散，更加平等，更加平等主义。互联网渠道看起来更像长尾销售分布，而目录渠道看起来则更像是传统的80/20。当研究人员更详细地查看数据时，他们发现，互联网渠道上的销售额分布更广，互联网渠道上销售的产品更加细分。与目录频道相比，这非常有趣，因为如前所述，我在一开始提到这项研究时，两个渠道上销售的产品类型是相同的，价格也是相同的。因此，在这种情况下没有供应方面的解释。

因此，并不是互联网渠道相比目录渠道提供了更多的产品种类，这两者都是相同的。然后研究人员进一步考虑，他们想到了另一种解释，如果您继续听，前文提到，这可能是一个需求方面的解释。因此，也许当您在网上购买时，您更容易找到那些更小众、不太受欢迎的产品。这正是这项研究的作者发现的情况。他们发现，在互联网渠道上，因为有各种工具，如评论和搜索能力等，这使得您更容易找到您在其他情况下可能不会遇到的东西。这不是非常有趣吗？长尾既是供应方面的故事。我可以提供更多的产品种类。

但这也是需求方面的故事。我更容易找到与我口味完全匹配的东西。这项研究的巧妙之处在于，办公室控制了供应端，他们发现互联网上的长尾销售比目录销售要多，原因就在这个需求端解释。

空间长尾与 Netgrocer

现在，关于长尾的最后一点讨论，到现在你可能已经牢固地记住了那个图表，X 轴是产品，Y 轴是流行度，头部是红色，尾部是橙色。现在，也许可以想想其他可以放在 X 轴上的东西，为什么它只能是产品。也许你现在正在想一些东西。让我抛出一个我特别感兴趣的东西，因为我经常看互联网卖家以及他们是如何做的。你也可以想象 X 轴不一定是产品，而是位置。因此，比如我是 diapers.com，我在洛杉矶、纽约市和旧金山销售了大部分产品，这些就是头部位置。但你知道吗？

我在一些奇怪的小镇上也卖了一些东西。我不应该说这个，因为也许有人住在那里。我在小镇上也卖了一些东西。但是，从整体来看，美国所有这些小镇加起来就是一个很大的数字。因此，长尾理论也可以应用于地理领域。我将从自己的研究中快速展示一个例子。因此，在这张幻灯片上，有一些我们在讨论过程中提到过的公司。在这个案例中，研究不是基于 Diapers.com，而是基于他们的朋友 Netgrocer.com。Netgrocer.com 是一家将各种超市里可能购买的杂货产品送货上门的零售商。

为了理解长尾理论是如何运作的，我们需要稍微思考一下在一个大型国家（无论是美国还是我们可能关注的其他国家）中，不同地点之间的相似性和差异性。现在，在左侧，我们有一张美国的物理地图，我们可以在地图上计算出芝加哥和洛杉矶之间的距离，大约是 2000 英里。我们还可以计算出芝加哥和斯普林菲尔德之间的距离，这个距离大约是 200 英里。因此，在物理距离方面，芝加哥和洛杉矶相距甚远，但芝加哥和斯普林菲尔德却相当接近。然而，当我们考虑社会距离时，居住在芝加哥的人可能比居住在像斯普林菲尔德这样的小镇上的人更相似于居住在洛杉矶的人。

因此，如果我们观察像 diapers.com 或 netgrosser.com 这样的公司在全美范围内的销售情况，我们可能会发现什么？因此，我做了这件事，再次与我的同事 Jonghei 以及我在纽约大学的另一位朋友 Sam Wui 一起，我们发现的结果非常有趣。我们发现，互联网零售商的销售也以长尾模式在地理上分散开来。最初，销售在像纽约、芝加哥、洛杉矶和旧金山这样的大市场起飞。但过了一段时间，你开始注意到其他位于尾部的位置，这些位置在物理距离上可能相隔很远，但这些市场的客户却具有相同的特征。

因此，这只是我在自己的研究中发现的一个想法，这一发现值得关注，因为它与这个非常关键的概念——长尾效应——有关。

对长尾理论的两项批评
